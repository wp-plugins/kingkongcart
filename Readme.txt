== Description ==

쿱 맴버스는 워드프레스협동조합에서 제공하는 한국형 소셜로그인 / 회원가입 플러그인 입니다.

공식홈페이지 http://www.wpcoop.kr

== Installation ==

1. 워드프레스 공식 홈페이지에서 coop-members 를 다운로드 받습니다.
2. 대시보드에서 설치를 진행하시려면 플러그인검색 에서 coop-members를 검색 하시면 됩니다.
3. 파일로 설치 하실 경우 wp-admin/plugins/coop-members 폴더를 생성하신 후 압축을 풀어 주시면 됩니다.
